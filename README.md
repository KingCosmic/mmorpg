# What is this?
This is ment to be a starting point for a mmorpg project.

# How to use?
The client and server are kept different so that they can be deployed differently. (for example you may want to wrap the client in electron)

## Client
cd client
npm install
npm start

## Server
cd server
npm install
npm run build
npm start
